O que é o StampForge

O StampForge é uma aplicação de design de carimbos para impressão 3D FDM que corre inteiramente no browser — sem instalação, sem servidor, sem conta. É um único ficheiro HTML.

Podes desenhar à mão livre ou com formas geométricas, importar SVGs vectoriais ou imagens raster, e exportar directamente para STL (para slicer) ou G-code (para imprimir sem slicer).

Requisito de browser
Usa Chrome ou Edge para todas as funcionalidades incluindo impressão USB via Web Serial. Firefox e Safari funcionam para desenho e export de ficheiros, mas não suportam USB.

Quick Start

1

Abrir o ficheiro

Abre stampforge_v31_preview.html no Chrome ou Edge. Não precisas de servidor — funciona directamente do sistema de ficheiros.

2

Definir dimensões físicas

No painel direito define Width e Height em milímetros (ex: 40 × 40 mm para um carimbo pequeno). Mantém a resolução em 600px para uso geral, ou sobe para 800–1024px para maior detalhe.

3

Desenhar o design

Selecciona uma ferramenta na barra do topo e desenha na canvas branca. O lápis (P) é o ponto de partida. Usa simetria para designs centrados.

4

Verificar o toolpath

Clica PREV para ver a visualização por camadas — confirma que o contorno está limpo e os detalhes têm espessura suficiente para o teu nozzle.

5

Exportar e imprimir

Clica GCODE para exportar directamente, ou STL para importar no teu slicer favorito. Para impressão directa por USB clica USB.

Cor e tamanho do brush

O selector de cor (quadrado colorido no header) abre o color picker nativo do browser. O campo numérico ao lado controla o tamanho do brush em pixeis.

Undo / Redo

Até 30 passos de undo com Ctrl+Z e redo com Ctrl+Y. Também disponíveis nos botões ↩ ↪ da barra de ferramentas.

Simetria

Os dois botões de simetria na barra de ferramentas activam eixos de espelhamento em tempo real. Cada traço é replicado automaticamente para o lado oposto.

BotãoEixoEfeito
⇔Vertical (centro)O que desenhas à esquerda aparece à direita
⇕Horizontal (centro)O que desenhas em cima aparece em baixo

Podes activar os dois em simultâneo para simetria em 4 quadrantes. As guias laranja tracejadas mostram os eixos activos na canvas.

Dica
Simetria vertical + horizontal é ideal para designs de carimbo centralizados como monogramas, flores, mandalas simples.

Importar SVG

Clica o botão ⬆SVG na barra para abrir o modal de import. Selecciona um ficheiro .svg.

Modos de import

⬆ Importar SVG

Vector layer permanente

  1. O SVG fica numa layer separada acima do canvas bitmap
  2. Aparece o badge laranja SVG VECTOR na canvas
  3. Todos os exports usam o SVG de alta resolução merged com o bitmap
  4. Podes desenhar com as ferramentas bitmap por baixo
  5. Para remover: clica o badge SVG VECTOR

🖼 Converter Bitmap

Rasterização directa

  1. Aparece um slider extra B&W threshold (10–245)
  2. Ajusta o threshold para controlar onde corta o preto/branco
  3. Ao confirmar, o SVG é rasterizado para o canvas bitmap
  4. O estado SVG é descartado — podes editar livremente
  5. Usa o eraser para limpar áreas indesejadas
Layers no canvas
Quando o modo Vector está activo existem três layers: bitmap (fundo, editável) → SVG (meio, vector persistente) → overlay (guias de simetria). O export merge automaticamente as duas layers de conteúdo.

Importar Imagem

O botão 🖼 (ao lado do ⬆SVG) importa qualquer imagem raster — PNG, JPG, GIF, WebP. A imagem é escalada proporcionalmente para caber na canvas e composta sobre o desenho existente.

Usa o eraser para limpar fundos e áreas indesejadas. Aumenta o parâmetro Simplify se o contorno gerado tiver demasiados pontos após importar imagens com muito detalhe.

Tamanho Físico & Resolução

ParâmetroDefaultNotas
Width (mm)60Largura física do carimbo em milímetros
Height (mm)60Altura física do carimbo em milímetros
Resolution600 pxResolução do canvas — afecta qualidade de contorno

Ao alterar a resolução, o desenho existente é reescalado para o novo tamanho. Usa 800–1024 px para designs com texto pequeno ou linhas finas.

Regra prática
Traço mínimo ≈ (dimensão mm / resolução px) × nozzle (mm). Com 60mm / 600px / 0.4mm nozzle o traço mínimo visível é ~0.04mm por pixel — muito inferior ao nozzle. Em prática, traços < 2–3 pixeis podem não imprimir bem.

Mirror for stamp
Este checkbox é essencial. O carimbo precisa de ser impresso espelhado para que, ao ser usado, a imagem apareça correcta no papel. O STL e G-code exportados já incluem este espelhamento. O export SVG não é espelhado — representa o design original.

Export STL

Clica STL no header para exportar um ficheiro STL binário watertight. Importa no teu slicer (PrusaSlicer, Bambu Studio, Cura, OrcaSlicer, etc.).

  • A face de design é o topo do modelo (Z+)
  • Se Mirror for stamp estiver activo, a geometria já está espelhada — não mirrors no slicer
  • A base rectangular (se incluída) tem as dimensões físicas definidas
  • Alturas: base = Base Thickness, relevo = Design Layers × Layer Height
No slicer
Imprime com 0% infill na base e 100% na face de design para melhor qualidade de superficie. O G-code gerado pelo StampForge já aplica 100% de infill no design.

Export G-code

Exporta G-code pronto para imprimir — sem slicer. Selecciona o perfil de impressora antes de exportar.

  • Infill rectilíneo 100% na camada de design
  • Perímetros configuráveis (default 2)
  • Retracção, fan, temperaturas do perfil seleccionado
  • G-code de início e fim completo (homing, heating, purge line)
  • Compatível com Marlin e Klipper

Export SVG

Exporta um SVG vectorial limpo com paths em milímetros, viewBox correspondente ao tamanho físico.

  • Coordenadas em mm, não espelhado (design face original)
  • Se uma layer SVG estiver activa, é merged com o bitmap na exportação a 4× resolução
  • Usa para corte laser, vinil, ou edição em Inkscape / Illustrator
  • fill-rule="evenodd" para furos correctos (letras com buracos, etc.)

Usa o layer preview para verificar se o design tem espessura suficiente, se os detalhes finos vão ser impressos, e se as ilhas ficam correctamente conectadas.

Impressão USB Directa

Requer Chrome ou Edge com Web Serial API. Funciona com Prusa MK3S por USB.

1

Ligar a impressora

Conecta a Prusa MK3S ao computador por USB. Aguarda que o driver instale (o USB CDC já está no stack da impressora — não precisas de drivers extra).

2

Clicar USB no StampForge

Clica USB no header. Abre o modal de impressão USB.

3

Conectar a porta

Clica CONNECT USB. O browser mostra um selector de porta serial — selecciona a porta da impressora.

4

Imprimir

Clica PRINT. O G-code é transmitido linha a linha com handshake ok — a impressora confirma cada comando antes do próximo ser enviado.

Workflows Recomendados

Carimbo de raiz

  1. Define dimensões (ex: 40 × 40 mm)
  2. Activa Mirror for stamp
  3. Activa simetria vertical para design centrado
  4. Desenha com pencil ou formas geométricas
  5. Ajusta Simplify e Curve smooth
  6. Clica PREV para verificar toolpath
  7. Export GCODE → imprimir

A partir de SVG (logo)

  1. Clica ⬆SVG → selecciona ficheiro
  2. Ajusta Scale e Position no modal
  3. Se quiseres editar: Converter Bitmap + threshold
  4. Se queres qualidade máxima: Importar SVG
  5. Desenha detalhes extra no bitmap se necessário
  6. Export STL ou GCODE

A partir de imagem raster

  1. Clica 🖼 → selecciona PNG/JPG
  2. A imagem é escalada para a canvas
  3. Usa eraser para limpar o fundo
  4. Aumenta Simplify se houver demasiados pontos
  5. Verifica no PREV
  6. Export

SVG + detalhes desenhados

  1. Importa SVG em modo Vector
  2. Selecciona pencil e desenha detalhes no bitmap
  3. O export combina as duas layers automaticamente
  4. Remove o SVG a qualquer altura clicando o badge laranja
